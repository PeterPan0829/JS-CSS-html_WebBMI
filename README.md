## BMI數值說明
* BMI ＜ 18.5 = 過瘦
* 18.5 ≦ BMI ＜ 24 = 理想
* 24 ≦ BMI ＜27 = 過重
* 27 ≦ BMI ＜30 = 輕度肥胖
* 30 ≦ BMI ＜35 = 中度肥胖
* 35 ≦ BMI = 重度肥胖
